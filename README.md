# AMD-Stock_Market_Analysis_Project_Python-
Stock Market Analysis

This project focuses on analyzing the historical stock performance of AMD – Advanced Micro Devices, a leading company in the semiconductor industry. The goal is to uncover market trends, price movements, volatility, and investment insights using data-driven techniques.

The project demonstrates skills in Python, Data Analysis, Visualization, and Financial Metrics by exploring AMD’s market behavior over time.

Load and clean historical AMD stock data
Perform exploratory data analysis (EDA)
Visualize stock price trends and volume changes
